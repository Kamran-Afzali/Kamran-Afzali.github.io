* Locke → Hume → Kant → Wittgenstein → Quine → Davidson → Gadamer → Kripke

* James → Bergson → Whitehead → Dennett → Metzinger → Churchland → Barad

* Rousseau → Kant → Mill → Marx → Beauvoir → Foucault → Sen → Butler → Mbembe
